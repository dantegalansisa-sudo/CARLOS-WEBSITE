# Compraventa Cordero

Website profesional para **Compraventa Cordero**, negocio de compra y venta de pasolas, motos, prendas de joyeria, casas y propiedades ubicado en Azua de Compostela, Republica Dominicana.

## Estructura del Proyecto

```
CARLOS-WEBSITE/
├── index.html
├── assets/
│   ├── css/
│   │   └── styles.css
│   ├── js/
│   │   └── main.js
│   └── images/
│       ├── logo.png
│       └── fundador.jpg
└── README.md
```

## Imagenes Requeridas

Para que el sitio se vea completo, agrega los siguientes archivos:

- **Logo:** Coloca el archivo `logo.png` en `/assets/images/`. El logo debe tener fondo negro con letras doradas.
- **Foto del fundador:** Coloca el archivo `fundador.jpg` en `/assets/images/`.

## Tecnologias

- HTML5, CSS3, JavaScript (vanilla)
- Google Fonts (Cormorant Garamond, Montserrat)
- Sin frameworks ni dependencias externas

## Uso Local

Abre `index.html` directamente en tu navegador. No se requiere servidor ni instalacion.

## Deploy en Railway

1. Crea un nuevo proyecto en [Railway](https://railway.app)
2. Conecta tu repositorio de GitHub o sube el proyecto
3. Configura como **Static Site**
4. Establece el **Root Directory** como `/` (raiz del proyecto)
5. Railway detectara automaticamente que es un sitio estatico y lo desplegara
